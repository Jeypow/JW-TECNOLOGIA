As Evoluções Web

WEB 2.0

A web 2.0 vem com evoluções quando comparadas com a web 1.0, trazendo novas funcionalidades para os usuários. Sendo lembrada como a “Web colaborativa”, a web 2.0 foi marcada pelo avanço na comunicação dos sites, blogs e fóruns, que agora conseguiam ter interação humana, uma vez que os usuários conseguiam colocar comentários, compartilhamentos, envio de imagens e outros arquivos ficaram possíveis na web 2.0. Outra característica da web 2.0 foi o uso mais amplo do CSS (que já era utilizado na web 1.0), onde possibilitou a criação de páginas mais bem elaboradas. Nessa era, a produção de conteúdo pelos internautas ganhou espaço por meio da possibilidade de inserção de comentários em páginas de blogs e se tornou o centro da experiência web com a chegada das redes sociais.
