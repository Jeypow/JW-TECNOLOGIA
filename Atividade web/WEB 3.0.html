As Evoluções Web

WEB 3.0

A web 3.0 tem foco na coleta de dados e inovação na transferência de dinheiro sem que seja necessário a presença de terceiros. Na web 3.0 o foco não somente é as interações sociais entre os usuários, mas também as interações econômicas. Os usuários da web 3.0 tem personalização individual dependendo de seu uso na internet, já que com a coleta de dados feitas na web 3.0 é possível que conteúdos específicos sejam recomendados para os usuários dependendo das necessidades dele naquele momento. Todo esse movimento de marketplace fez com que muitos resumissem a web 3.0 à web do marketing. Porém, como já mencionado, a web 3.0 trouxe inovações também no âmbito econômico, sendo agora possível possuir e comercializar moedas virtuais como as bitcoins, os tokens (NFTs) e o blockchain.
